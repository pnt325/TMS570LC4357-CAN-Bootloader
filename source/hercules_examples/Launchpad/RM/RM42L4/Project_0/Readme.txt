The objective Project 0 is to build a simple example for the purpose of verifying that you have 
a working development environment. The project itself will do something very simple: Blink an LED.

HW: TI Hercules RM42x Launchpad
Code Composer Studio (CCS): v9.x 
Compiler Version: TI V18.12.1.LTS
HALCoGen: 4.07.01


--------------------------------------------------------------------------------------------------
Import the project to CCS
--------------------------------------------------------------------------------------------------
1. Open CCS
2. Make sure you are in the "CCS Edit" perspective
3. From the CCS Menu, select Project->Import CCS Projects...
4. When the "Import CCS Eclipse Projects" window opens, Browse the installation folder and Select 
   the project: RM42x_LaunchPad_Project0
